sun-type6-keyboard
==================

sun type6 keyboard & mouse as USB HID device

| sun type6 keyboard | arduino pro micro   |
|--------------------|---------------------|
| red                | +5v                 |
| black              | GND                 |
| blu(TX)            | 7(software RX)      |
| brown(RX)          | 8(software TX)      |
| green(mouse)       | TBD                 |
